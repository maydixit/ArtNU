# Renaissance Go 

App for a private party : Find paintings & unlock neural styles. 

Features:
- QrCode reading
- Screenshots
- Neural style transfer  (based on [Fritz.ai](https://app.fritz.ai/))
- 3D version w/ Google cardboard
